# DiveAnalyzer

A small Python project for analyzing and visualizing dive data.

## Features

- Load dive data from CSV
- Generate a dive report
- Visualize depth and temperature over time
- Save plots as PNG files

## Tech Stack

- Python
- Pandas
- Matplotlib
- Git
